respread.DynamicNode.child_factory#

DynamicNode.child_factory(name: str) → Callable#

Function that defines result of calling non-existent attr, should be overridden.

If the object does not have an attribute name and name is in self.get_derived_children(), this method will be called on attribute lookup. Concrete subclasses should return a callable for the child of name. The callable will be bound before it is returned.

Parameters#

namestr

Name of requested child attribute
